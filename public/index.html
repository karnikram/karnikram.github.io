Research
Marker-less Cross-Sensor Extrinsic Calibration
Karnik Ram*, Ganesh Iyer*, Krishna Murthy, KM Krishna

Working on an automatic and markerless approach for calibrating the extrinsic parameters between a lidar and a camera, within the context of a self-driving car, through non-linear photometric error minimization. The approach does not use feature correspondences, making it invariant to dynamic scenes.

In progress.
Projects
Automated Stock Counting Using a Quadcopter
Karnik Ram, Harish S, Apeksha Avinash  |  Winter 2016

Developed a visual odometry module based on optic flow for the localization of a custom-built quadcopter and incorporated it into the PX4 navigation stack, enabling autonomous indoor navigation. All the computations were performed on-board, on an Odroid XU4. A stock counting module was implemented using ArUco markers.
